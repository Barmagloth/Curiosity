# Exp0.7 — Soft Gating: от бинарного переключателя к плавному подмешиванию

## Контекст

Exp0.4: одиночный residual = оракул на чистых данных.
Exp0.5: оракул ломается при шуме (corr 0.90 → 0.54).
Exp0.6: бинарный переключатель работает, но alias — пограничный случай, и "всё или ничего" грубовато.

Следующий шаг: **плавные веса** вместо switch. Каждый эксперт получает вес пропорционально своей "полезности".

---

## Эксперты

| # | Имя | Сигнал | На чём считается |
|---|-----|--------|------------------|
| E_resid | Residual | abs(upsample(coarse) - target) | target-grid → tile mean |
| E_var | Variance | local_var(coarse, 3×3) | coarse-grid |
| E_hf | HF energy | Laplacian | coarse-grid |
| E_cost | Окупаемость | depth_penalty * boundary_penalty | meta (не из данных) |
| E_probe | Exploration | давность + неопределённость | meta |

E_cost и E_probe — не "про данные", а про бюджет и exploration. Их вес фиксирован (не гейтится), добавляется отдельно.

Гейтятся только E_resid, E_var, E_hf.

---

## Гейт (скучный бухгалтер)

### Диагностика здоровья эксперта i

На probe-наборе (5-10% бюджета) вычисляем:

```
c_i = corr(E_i, gain_probe)   # предсказательная сила
s_i = 1 - (Δnodes_i / N_tiles) # стабильность под jitter
```

### Вес

```
raw_i = α * clip(c_i, 0, 1) + β * s_i
w_i = softmax(raw_i / temperature)
```

- α = 0.7, β = 0.3 (корреляция важнее стабильности)
- temperature = 0.3 (достаточно резкий softmax)
- EMA smoothing: `w_i = λ * w_i_new + (1-λ) * w_i_prev`, λ = 0.3
- Гистерезис: доминирующий эксперт меняется только если разница > 0.15

### Итоговая ρ

```
ρ = Σ w_i * norm(E_i) + w_cost * E_cost + w_probe * E_probe
```

w_cost и w_probe — фиксированные гиперпараметры (не гейтятся).

---

## Аварийный режим

Если **все** c_i < 0.3 (все эксперты плохи):

1. Бюджет уходит в probe (50% вместо 5-10%)
2. Приоритет на крупные структуры (low-freq residual)
3. Garbage-detector ужесточается
4. Сплиты минимизируются до "только самое очевидное"

---

## Известные режимы отказа

| Проблема | Симптом | Реакция гейта |
|----------|---------|---------------|
| Шумный coarse | corr(resid)↓, FSR↑ | w_resid↓, w_var↑ |
| Текстурный терроризм | FSR↑, ROI_q10↓, карта "кипит" | w_hf↓, w_cost↑ |
| Blur | HF→0, variance→0 | w_resid↑ (единственный живой) |
| Alias/moire | HF ловит ложные частоты | w_hf↓ (через stability) |
| Все плохи | все c_i < 0.3 | аварийный режим |

---

## Метрики Exp0.7

Те же что в Exp0.4/0.5 + дополнительно:

- **w_i per step**: веса экспертов на каждом уровне
- **c_i per step**: корреляции экспертов с gain
- **gate_entropy**: -Σ w_i log(w_i) — насколько "уверен" гейт
- **mode_switches**: сколько раз доминирующий эксперт менялся за прогон
- **emergency_rate**: доля шагов в аварийном режиме
